insertAfterAllTasks

GradleTask.Builder insertAfterAllTasks(TaskName target)
Content copied to clipboard

Inserts tasks which will precede the new one.

Unlike insertAfterTask(), this method will depend the new task on every task with such name in the project (i.e. the tasks of the root project and all the subprojects).

If a certain project does not have a task with the specified name, no action is performed for that project.

This method does not guarantee that the task will be included into a standard Gradle build.

Invocation of this method may substitute the invocation of insertAfterTask or insertBeforeTask if it's guaranteed that at least one task with such name exists. Though the fallback is never handled and there is no guarantee that the task will get into the Gradle task graph.

Return

the current instance of Builder

Parameters

target

the name of the tasks, serving as "after" anchor
