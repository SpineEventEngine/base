ExcludeInternalDoclet

public class ExcludeInternalDoclet

Extension of Standard doclet, which excludes Internal-annotated components.

Use it to generate documentation for audience, that should not know about Internal-annotated components.

Works by pre-processing a RootDoc. The doclet creates new RootDoc, that does not contain Internal-annotated components and further generates documents.

You can use the non-standard doclet by specifying the following Javadoc options:

  • doclet io.spine.tools.javadoc.ExcludeInternalDoclet;
  • docletpath classpathlist (The path to the doclet starting class file).

Call it with Javadoc tool like this: 

javadoc -doclet io.spine.tools.javadoc.ExcludeInternalDoclet
-docletpath "classpathlist" ...
