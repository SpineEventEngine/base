valueIsPresent

BooleanExpression valueIsPresent(Expression<? extends Object> valueAccess)
Content copied to clipboard

Produces an expression which tests if a single value of the field is present.

If the field is singular, the expression is equivalent (but not the same) as the one produced by invocation. If the field is a collection however, the produced expression only tests a single value (i.e. element of the collection).

Return

the boolean expression testing if the value is non-default

Parameters

valueAccess

the value to be tested
