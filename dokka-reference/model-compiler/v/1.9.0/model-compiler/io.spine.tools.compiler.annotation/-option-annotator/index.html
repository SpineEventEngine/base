OptionAnnotator

public abstract class OptionAnnotator<D extends GenericDescriptor> extends Annotator

An Annotator which annotates Java sources basing on a given Protobuf option.

Subtypes are purposed for different Protobuf option types such as FileOptions, MessageOptions etc.

Depending on the option type, an annotator manages a corresponding Protobuf descriptor (e.g. FileDescriptorProto for FileOptions).
