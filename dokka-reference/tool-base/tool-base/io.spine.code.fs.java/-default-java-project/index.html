A default directory structure for a Spine-based Java project.

The project structure is based on the standard Maven/Gradle project conventions, with the following directories under the project root:

  • src/main — manually written code production code, with java and proto sub-directories.
  • src/test — the code of tests.
  • generated — computer-generated code, with sub-directories:
    • java — the code generated by the Protobuf Compiler.
    • gRPC — the code generated by the gPRC Protobuf Compiler Plug-in.
    • spine — the code generated by the Spine Model Compiler.
  • .spine — temporary build artifacts directory used by the Spine Model Compiler.
