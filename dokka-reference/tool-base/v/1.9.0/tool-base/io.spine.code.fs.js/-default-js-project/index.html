DefaultJsProject

public final class DefaultJsProject extends DefaultProject

A default directory structure for a Spine-based JavaScript project.

The project structure reflects the conventions currently used in Spine, and contains at least the following directories:

  • build
    • descriptors
      • main
        • known_types.desc — descriptors for "main" source set.
      • test
        • known_types.desc — descriptors for "test" source set.
  • generated — the JS code, generated by Protobuf JS compiler.
    • main
      • js — generated messages from "main" source set.
    • test
      • js — generated messages from "test" source set.

Other directories (like source code directory) may also be present in the project, but their location is currently not standardized and thus is not described by this class.
