DefaultProject

public abstract class DefaultProject

This class represents a default directory structure for a Spine-based project of any language.

The descendants of the class contain the language-specific project structures.

The DefaultProject helps resolving names of the directories and files under the project directory. It is expected that for most projects, the default values of paths remain unchanged.
