Configures strongly-typed fields generation for messages that represent an entity state.

Example: 

generateFor messages().entityState(), markAs("com.some.custom.Field")

Configures the strongly-typed fields generation for messages declared in files matching a given pattern.

Example: 

generateFor messages().inFiles(suffix: "messages.proto"), markAs("com.some.custom.Field")
Content copied to clipboard

The statement above enables strongly-typed fields generation for all messages residing in files with names ending with messages.proto and marks the generated fields as some.custom.Field.

It is expected by the Spine routines that the type passed to the markAs method is a public non-final descendant of io.spine.base.SubscribableField and has a public constructor accepting a single Field argument.

In case the field generation is configured for event and rejection messages, the passed field type should inherit from io.spine.base.EventMessageField.

The configuration may be applied multiple times to enable code generation for multiple file patterns.
