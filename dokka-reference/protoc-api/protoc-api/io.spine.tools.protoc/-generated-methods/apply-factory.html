Configures method generation for messages declared in files matching a given pattern.

Sample usage is as follows: 

applyFactory "io.spine.code.CustomMethodFactory",messages().inFiles(suffix: "events.proto")
Content copied to clipboard

The statement in the example above configures all message types declared in a file which name ends with events.proto to use io.spine.code.CustomMethodFactory as a custom method factory. It is expected that io.spine.code.CustomMethodFactory is an implementation of the io.spine.tools.protoc.method.MethodFactory interface. Example of a possible implementation: 

// In io/spine/code/CustomMethodFactory.java:

package io.spine.code;

public CustomMethodFactory implements io.spine.tools.protoc.method.MethodFactory {

    public CustomMethodFactory() {
    }

   List<MethodBody> createFor(MessageType messageType) {
        // ...
    }
}

// In build.gradle:

// ...

modelCompiler {
    methods {
        applyFactory "io.spine.code.CustomMethodFactory", messages().inFiles(suffix: "events.proto")
    }
}

This method functions similarly to the applyFactory except the file in which the message type is defined does not matter.

applyFactory "io.spine.code.CustomMethodFactory", messages().uuid()
