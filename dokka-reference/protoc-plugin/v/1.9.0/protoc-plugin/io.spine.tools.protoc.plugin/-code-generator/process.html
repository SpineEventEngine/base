public final CodeGeneratorResponse process(CodeGeneratorRequest request)

Processes the given compiler request and generates the response to the compiler.

Each .proto file may cause none, one or many generated CodeGeneratorResponse.File instances.

Note: there are several preconditions for this method to run successfully:

  • since Spine relies on 3rd version of Protobuf, the Proto compiler version should be 3.* or greater;
  • there must be at least one .proto file in the CodeGeneratorRequest.

the response to the compiler
