static boolean ensureFile(File file)

Ensures that the given file exists.

Performs no action if the given file already exists.

If the given file does not exist, it is created along with its parent directories, if required.

If the passed File points to the existing directory, an IllegalArgumentException is thrown.

In case of any I/O issues, the respective exceptions are rethrown as IllegalStateException.

Return

true if the file did not exist and was successfully created; false if the file already existed

Parameters

file

a file to check

Throws

java.lang.IllegalArgumentException

if the given file is a directory

java.lang.IllegalStateException

in case of any I/O exceptions

Ensures that the file represented by the specified Path exists.

If the file already exists, no action is performed.

If the file does not exist, it is created along with its parent if required.

If the specified path represents an existing directory, an IllegalArgumentException is thrown.

If any I/O errors occur, an IllegalStateException is thrown.

Return

true if and only if the file represented by the specified path did not exist and was successfully created
