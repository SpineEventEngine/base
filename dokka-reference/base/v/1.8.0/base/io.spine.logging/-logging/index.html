Logging

public interface Logging
Content copied to clipboard

Utility interface for objects that require logging output.

Such an object needs to implement this interface and obtain a FluentLogger instance associated with the class of the object via the logger method.

In addition to this, the interface provides shortcut methods for the popular logging interface methods. These shortcut methods are named after the tracing level of Java Logging (such as _fine or _severe, and aliases for the levels popular from other logging frameworks (such as _debug or _error.
